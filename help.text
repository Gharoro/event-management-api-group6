1. Install the seuelize CLI
	npm install -g sequelize-cli

2. Create a database in pgAdmin named 'event_management_dev"

3. Run the migration
	npx sequelize-cli db:migrate

Following the above steps will generate all the tables related to event managers in the database you created.

To have express working, simply run 
    npm install


Also, create a .env file in the root directory and update it with the right information related to you (see .env-sample file)
